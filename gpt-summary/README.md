# 关于转弯 车道变化 【中文驾考手册p21】

## **“One-way grid system”**（城市常见双单行道路网）。

> - 一般原则（靠左进左）
> - 特殊情况（多左转车道）

实操原则总结

| **情境**      | **左转起点**   | **左转终点**       | **是否可多车道**       |
| ------------- | -------------- | ------------------ | ---------------------- |
| 🚗 双向 → 双向 | 最左车道       | 最左车道           | ❌ 不可                 |
| 🚙 双向 → 单行 | 最左车道       | 最左（靠中线）车道 | ✅ 若有导向箭头可多车道 |
| 🛣 单行 → 单行 | 任一左转专用道 | 对应目标车道       | ✅ 按箭头走，不可跨线   |
| 🚧 单行 → 双向 | 最左车道       | 最左车道           | ❌ 只有一条合法路径     |

> **一句话记忆法**
>
> > 🧭 **没有箭头 → 靠左进左。**
>
> > 🛣 **有箭头 → 按箭头走。**

------

## **🧠 总原则（所有转弯共通）**

> ✅ **“Closest to Closest” 靠近原则：**
>
> - 从你所在的**最靠近要去方向的那条车道**开始转弯，
>
> - 转入目标道路上**最靠近你转弯方向的那条车道**。
>
> 🚫 不允许跨越多条车道完成转弯（unless lane marking allows it）。

------

## **🟢 一、Two-way → Two-way（最常见）**



```
双向 → 双向
---------------------------------
← | 你所在车道 | →（对向）
         ↓ 左转
              ↙
---------------------------------
← | 目标道路 | →
```

**🚗 左转：**

- **从最左侧车道**（靠中线）开始；
- **转入目标道路的最左侧车道**；
- 如果想去右边车道，必须转完弯后再变道。

**🚗 右转：**

- **从最右侧车道**右转；
- **转入目标道路的最右侧车道**；
- 不得跨线进入中间或左车道。

✅ **口诀：**

> 左进左，右进右，转完才能换。



## **🟣 二、Two-way → One-way（题目考点）**

```
双向 → 单行（右行）
---------------------------------
← | 你所在车道 | →（对向）
         ↓ 左转
              ↙
---------------------------------
→ | → | → （单向）
```

**🚗 左转：**

- 从双向路的**最左车道**转入；
- 进入单行道的**最左车道（first lane）**；
- 转完后可变道。

✅ **口诀：**

> 两向转单向，左进最左道。

------

## **🔵 三、One-way → Two-way**

```
单向 → 双向
---------------------------------
→ → → 
       ↘
---------------------------------
← | →（双向）
```

**🚗 左转：**

- 从单行道的**最左侧车道**左转；
- 转入双向道路的**靠近中线的左侧车道**。

**🚗 右转：**

- 从单行道的**最右车道**右转；
- 转入双向道路的**最右车道**。

✅ **口诀：**

> 单向出，左右转仍遵“靠边原则”。

------

## **🟠 四、One-way → One-way**

```
单行 → 单行
---------------------------------
→ → → 
       ↘
---------------------------------
→ → → 
```

**🚗 左转：**

- 从单行道的**最左车道**；
- 转入目标单行道的**最左车道**；
- 如果有专用转弯线标，可直接进入指定车道。

**🚗 右转：**

- 从最右车道；
- 转入目标单行道的最右车道。

✅ **口诀：**

> 单向对单向，照常靠边转。

------

## **🟡 五、Divided Highway / Median（中央隔离带）**

**🚗 左转进入分隔道路：**

- 先驶入中央等待区（median or left-turn pocket）；
- 等安全时再并入右侧行车方向；
- **不能穿越整个隔离带一次性左转**（违法）。

------

## **🔴 六、U-turn（掉头）**

| **场景**    | **允许条件**         | **禁止条件**               |
| ----------- | -------------------- | -------------------------- |
| 双向道路    | 中央虚线处可掉头     | 实线中线 / “No U-Turn”标志 |
| 交叉口      | 绿灯左转信号下允许   | 红灯 / “No U-Turn”标志     |
| 单行道      | 仅能从**最左侧**掉头 | 不能掉头进逆向单行道       |
| 学区/住宅区 | 若无来车可掉头       | 有车辆距离≤200英尺禁止     |

✅ **口诀：**

> 掉头看线色，左边要虚可。

------

## **⚠️ 七、特殊例外情况**

| **情况**                                      | **特别规则**                       |
| --------------------------------------------- | ---------------------------------- |
| 🅿️ 有 painted turn guide arrow（地面导向箭头） | 按箭头进入指定车道，不可随意跨线   |
| 🚘 多车道左转（dual left turn lanes）          | 各走各道，内转内，外转外，禁止交叉 |
| 🚦 Protected left turn（带箭头信号）           | 只能按信号灯箭头方向完成转弯       |
| 🚫 “No turn on red” 标志                       | 即使无车，也不能在红灯右转         |
| 🟩 “Right turn on red” 允许                    | 停稳后看安全再右转                 |

------

## **📋 总表总结（超简版）**

| **起点道路**      | **目标道路**            | **左转进入**            | **右转进入** |
| ----------------- | ----------------------- | ----------------------- | ------------ |
| Two-way → Two-way | 最左车道 → 最左车道     | 最右车道 → 最右车道     |              |
| Two-way → One-way | 最左车道 → 单行最左车道 | 最右车道 → 单行最右车道 |              |
| One-way → Two-way | 最左车道 → 靠中线左车道 | 最右车道 → 靠右车道     |              |
| One-way → One-way | 最左车道 → 最左车道     | 最右车道 → 最右车道     |              |

------

**🧭 终极口诀总结**

> 🚦 “左转靠左，右转靠右；
>
> 最靠近的出发车道 → 最靠近的目标车道；
>
> 变道要另算，掉头要看线。”

------





# 低速场景限速

加州 DMV 笔试中最容易混淆的部分之一 —— 各种 **低速场景限速（15 mph / 25 mph / 35 mph）**。

## **🚗** **加州 DMV 常见限速总结表**

| **限速**   | **场景**                                                     | **说明**                                                     |
| ---------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **15 mph** | 🚧 **铁路道口 (railroad crossing)** — 如果在距离铁路 100 英尺内、且两边看不到 400 英尺 | 除非有闸门、红灯、旗手控制，否则必须降到 15 mph 以内。       |
| **15 mph** | 🏫 **盲校 (blind intersection)** — 看不到 100 英尺内的横向道路 | 适用于被建筑或树木遮挡视线的交叉口。                         |
| **15 mph** | 🚸 **巷道 / 小胡同 (alley)**                                  | 所有巷道、后巷都限制 15 mph。                                |
| **25 mph** | 🏘 **住宅区 / 商业区 (residential or business district)**     | 除非另有标示，否则默认 25 mph。                              |
| **25 mph** | 🏫 **学校区域 (school zone)** — 儿童在场或上下学时段          | 有标志或闪灯提示时，限 25 mph （部分标 15 mph 则依标识为准）。 |
| **25 mph** | 🚸 **儿童游乐区或公园周边**                                   | 若有儿童出现或标识提示，需减速至 25 mph 以下。               |
| **35 mph** | 🐎 **盲交叉路口 / 山区弯道带警示标志**                        | 若有 “Advisory Speed” 黄色牌，按其数值行驶。                 |
| **55 mph** | 🚛 **双向无分隔高速路 (two-lane undivided highway)**          | 加州默认限速 55 mph，除非另有标志。                          |
| **65 mph** | 🛣 **多车道公路 / 州际公路**                                  | 常见城市外环、州际主干道默认 65 mph。                        |
| **70 mph** | 🛣 **特定高速路段 (部分 Interstate)**                         | 若标志允许，可达 70 mph。                                    |

------

## **🧠** **记忆口诀：**

> 🚗「15 三小」：巷、小视角、小铁路
>
> 🏠「25 两区」：住宅、学校
>
> 🛣「55 65 70 三级路」：普通、高速、州际

------

## **⚠️** **常见陷阱题：**

| **题型**                           | **正确答案**          | **易错解释**            |
| ---------------------------------- | --------------------- | ----------------------- |
| Railroad crossing 无警示且视线受限 | **15 mph**            | 很多人误选 25 mph       |
| School zone 有闪灯                 | **25 mph 或标示速度** | 若标 15 mph 则依标志    |
| Residential area 没标志            | **25 mph**            | 默认限速                |
| Blind intersection                 | **15 mph**            | 仅当你看不到 100 英尺内 |

# 时间、距离类

## **一、时间类：秒 / “多久之前”相关**

### **1. 跟车 & 视线**

| **数字**     | **场景**                                           | **记忆方式**                                                 |
| ------------ | -------------------------------------------------- | ------------------------------------------------------------ |
| **3 秒**     | **正常跟车距离**，避免追尾                         | “3 秒跟车规则（3-second rule）”：选前车经过的固定点，数 “one-thousand-one…two…three”，自己到前说明太近。 |
| **≥4 秒**    | 雨天、雾天、夜间、后面拖挂车、跟大型车时的安全距离 | 题目出现 bad weather / towing / large vehicle → 把 3 秒加长到 4 秒或更多。 |
| **10–15 秒** | 视线向前看的距离，**避免 last-minute moves**       | “Look ahead 10–15 seconds”：城市大概一个街区，高速大约 1/4 英里。题里只要说 *avoid last-minute moves / see hazards early* 基本就是这个答案。 |

------

### **2. 打转向灯 / 车道变换**

| **数字**          | **场景**                                 | **记忆方式**                                                 |
| ----------------- | ---------------------------------------- | ------------------------------------------------------------ |
| **至少 100 英尺** | 城市/普通道路上转弯前打灯距离            | “Signal during the last 100 feet before turning”。           |
| **至少 5 秒**     | **高速上变道或驶入/驶出匝道时** 打灯时间 | 题目看到 *on a freeway, when changing lanes / exiting* → 选“signal for at least 5 seconds”。 |

------

### **3. 车灯开启时间**

| **数字**                            | **场景**           | **记忆方式**                                               |
| ----------------------------------- | ------------------ | ---------------------------------------------------------- |
| **日落后 30 分钟 ~ 日出前 30 分钟** | 必须开头灯的时间段 | “half hour after sunset, until half hour before sunrise”。 |

------

## **二、距离 / 视距类（英尺）：看不见多远就要干嘛**

### **1. 车灯和能见度**

| **数字**                         | **场景**                                 | **记忆方式**                                                 |
| -------------------------------- | ---------------------------------------- | ------------------------------------------------------------ |
| **看不清 1000 英尺**             | 太暗 / 雾 / 雨，必须开灯                 | 法规里写：看不清 1000 ft 人或车，就要用头灯。                |
| **500 英尺（会车）**             | 对向有车在 500 ft 内，要从远光换近光     | “Dim high beams within 500 ft of oncoming traffic”。         |
| **300 英尺（跟车）**             | 跟在别人后面小于 300 ft，要用近光        | “within 300 ft of a vehicle you are following”。             |
| **看不见 200 英尺的山路 / 弯道** | 窄山路、视线差弯道上，用喇叭提醒对向车辆 | DMV 写明：在窄小山路上，若不能看到至少 200 ft 远，要用喇叭提醒。 |

------

### **2. 铁路道口 / 铁路相关**

| **数字**                    | **场景**                                                     | **记忆方式**                                                 |
| --------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **15–50 英尺**              | 有 STOP 义务的车辆在铁路道口前停车距离                       | 题目写 *stop not less than 15 feet nor more than 50 feet from the nearest rail*。 |
| **100 英尺内，限速 15 mph** | 在铁路道口 100 ft 内，**视线不足 400 ft** 时的限速（乘用车） | 经常以：“Within 100 feet of a railroad crossing where you cannot see the tracks for 400 feet…” → speed limit 15 mph 出现。 |

------

## **三、停车 / 站距类（英寸 & 英尺）**

### **1. 与路缘（curb）的距离**

| **数字**        | **场景**                             | **记忆方式**                                  |
| --------------- | ------------------------------------ | --------------------------------------------- |
| **18 英寸以内** | 平行停车时，车轮与路缘的最大允许距离 | “parallel and within 18 inches of the curb”。 |

------

### **2. 不得停车的典型“禁止距离”**

这些数字题在 DMV practice 里非常常见，背下来很有用：

| **数字**        | **位置**                                                     | **说明**                                                     |
| --------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **15 英尺**     | **消防栓（fire hydrant）** 前                                | 禁止停车。                                                   |
| **20 英尺左右** | 交叉路口 / 人行横道附近（有的题会说 crosswalk at an intersection 20 ft） | 具体题目要看选项，但“离人行横道 / intersection 太近”一定不能停。 |
| **50 英尺**     | 铁路道口（railroad crossing）附近禁止停车距离                | “within 50 feet of the nearest rail” 不能停车。              |

> 小记忆法：15 – 20 – 50

------

## **四、再帮你做个“超简版口诀”**

- **10–15 秒**：视线向前看，避免 last-minute moves。

>
> **10–15 秒看远，**
>
> **100 ft 前开始打灯转弯，**
>
> **200 ft 山路看不见要按喇叭，**
>
> **500 / 300 ft 关远光，**
>
> **18 in 贴路缘，**
>
> **15 / 20 / 50 ft 别乱停车。**

我们来逐句拆解，为什么 **第一个选项才是对的**，而第二、第三个虽然“看起来有道理”，却都是**不符合加州法律的误导答案**。

> It is illegal to leave a child age six or younger unattended in a vehicle on a hot day.

**✅ 正确答案：**

> **Even if they are secured in a child passenger restraint system.**
>
> （即使孩子系着安全带，也违法。）

**🔍 理由：**

加州法律（California Vehicle Code §15620）明确规定：

> You may **not** leave a child age 6 or younger unattended in a motor vehicle,
>
> **even if** the child is properly restrained,
>
> when:
>
> - the keys are in the ignition, **or**
>
> - the engine is running, **or**
>
> - there is a significant risk to the child’s health or safety (例如炎热天气)。

🌡️ “Hot day” 属于「significant risk to health or safety」情形，因此无论是否有安全座椅，都**绝对违法**。

------

## **❌ 错误选项一：**

> **If they are supervised by a person twelve years of age or older.**

**🔍 理由：**

这句话本身是**法律允许的例外**，但前提是：

> 没有任何危险情形（如炎热天气、车窗关闭、高温）。

在本题中已经明确说是 “**on a hot day**”，也就是**高温危险环境**，

👉 所以即便有 12 岁以上的人陪伴，也**仍然违法**。

📘 DMV 手册原文：

> “A child may be left under the supervision of a person age 12 or older **only when it is safe to do so**.”

------

## **❌ 错误选项二：**

> **Only if the key is in the ignition.**

**🔍 理由：**

这是部分情况的限制条件之一，但**不是唯一条件**。

题目里写的是 “on a hot day”，重点在于 **heat danger**，和钥匙是否插在点火口无关。

即使钥匙拔掉，只要天气炎热、存在健康风险，仍属违法。

------

## **✅ 总结对比表：**

| **情境**               | **是否违法** | **备注**                         |
| ---------------------- | ------------ | -------------------------------- |
| 🔥 炎热天气（hot day）  | ✅ 违法       | 即使有安全座椅、车窗开缝，都违法 |
| 🔑 钥匙插在点火口       | ✅ 违法       | 存在安全风险                     |
| 👶 儿童无人看管         | ✅ 违法       | 不论天气                         |
| 👧 有 ≥12岁监督人且安全 | ❌ 不违法     | 仅当环境安全、无风险             |

------

## **🧠 一句话记忆：**

> ❗ **6岁以下儿童，除非有 ≥12岁看护且环境绝对安全，否则一律违法。**
>
> 🔥 **天气热 = 绝对违法，不论任何情况。**

------



# 驾照类型

这类题目确实是加州 DMV 笔试（Class C 驾照题库）中非常常见的「**隐藏型知识点**」：

它在手册正文里**没有完整表格列出**，而是**零散藏在 “License Classes” 章节**（一般在官方PDF靠前几页的小段落中）。

很多人读手册时会直接略过。

## **🧩 正确答案**

> ✅ **A three-axle vehicle if the Gross Vehicle Weight is less than 6,000 pounds.**

## **🧠 解释详解**

加州 DMV 的 **Class C 驾照（普通乘用车类）** 允许你驾驶的车辆包括：

| **允许驾驶**                        | **限制说明**                                            |
| ----------------------------------- | ------------------------------------------------------- |
| 🚗 **2轴车辆（two-axle vehicle）**   | 任何普通轿车、小卡、小货车。                            |
| 🚙 **3轴车辆（three-axle vehicle）** | ✅ **仅当总重量（GVWR）< 6,000 磅（≈ 2.7 吨）**。        |
| 🚚 **拖挂车辆**                      | 允许拖 1 个总重 ≤ 10,000 磅的拖车（如小拖车、小房车）。 |
| 🚛 **房车/露营车（housecar）**       | 长度 ≤ 40 英尺。                                        |
| 🏍 **摩托车/摩托助动车**             | 仅限摩托助动车（moped），不能开摩托车。                 |

------

## **🚫 不允许的（Class C 不包括）**

| **禁止驾驶**                                   | **原因**                   |
| ---------------------------------------------- | -------------------------- |
| ❌ 任何 3 轴车辆（如重型卡车）**超过 6,000 磅** | 需要 Class B 商业驾照。    |
| ❌ 拖 2 个拖车                                  | 仅 Class A CDL 可行。      |
| ❌ 载 10+ 乘客车辆（如小巴）                    | 需 Passenger Endorsement。 |

## **💬 题目逻辑分析**

> With a Class C driver license, a person may drive:
>
> - (A) A three-axle vehicle if the GVW < 6,000 lb ✅
>
> - (B) Any three-axle vehicle ❌（太宽泛）
>
> - (C) A vehicle pulling two trailers ❌（不允许）

**考点关键词：**

- 「**three-axle vehicle**」→ 限重6,000 lb
- 「**two trailers**」→ 只能 CDL-A
- 「**Class C license**」→ 普通非商用驾照

------

## **📘 补充资料来源**

> California Driver Handbook (2025) — Chapter 1: “License Classes”
>
> *A Class C license allows you to drive a 2-axle vehicle with a GVWR of 26,000 pounds or less, a 3-axle vehicle weighing 6,000 pounds or less, and a housecar 40 feet or less in length.*

# 关于天气

**Hydroplaning（车胎打滑 / 水上滑行）** 属于加州 DMV 手册中的**“恶劣天气与防御性驾驶”章节（Defensive Driving / Road Conditions）**，

我来帮你系统补全这一整块内容，包括 Hydroplaning、打滑、雨天/雾天/冰雪驾驶等所有**隐藏物理类知识点**👇

## **🌧️ DMV 手册隐藏知识点 · 天气与路况篇（Weather & Road Condition Traps）**

------

## **🚙 1️⃣ Hydroplaning（车胎水上滑行）**

| **项目**     | **说明**                                                     |
| ------------ | ------------------------------------------------------------ |
| **定义**     | 当车速过快时，轮胎下的水来不及排出，车辆漂浮在水面上，轮胎失去与地面的接触。 |
| **发生速度** | 🚫 **超过 35 mph** 后风险显著上升。                           |
| **主要原因** | - 雨后路面积水<br>- 胎纹浅或磨损<br>- 气压不足<br>- 高速行驶             |
| **危险表现** | 转向失灵、刹车无效、车辆“漂浮”感。                           |
| **应对方式** | ✅ **松开油门，直握方向盘**，切勿急刹或猛打方向。⚠️ 若有自动巡航，**立即关闭 Cruise Control**。 |
| **预防方法** | - 降速（<35 mph）- 确保胎纹≥1/32英寸- 雨天提前减速- 避开积水带 |

📘 *来源: California Driver Handbook 2025, Chapter “Handling Emergencies” & “Vehicle Condition”*

------

## **🧊 2️⃣ Skidding（打滑）**

| **类型**     | **原因**          | **正确做法**               | **常见错误**         |
| ------------ | ----------------- | -------------------------- | -------------------- |
| **加速打滑** | 在湿滑/冰面急加速 | 松开油门、不要刹车。       | 踩刹车。             |
| **刹车打滑** | 急刹或ABS启动     | 轻抬刹车，保持方向。       | 持续猛踩。           |
| **转向打滑** | 方向过猛          | **顺着车尾方向打方向盘**。 | 反打导致甩尾更严重。 |

------

## **🌧️ 3️⃣ 雨天驾驶（Rain Driving）**

| **知识点** | **正确做法**                                                 | **错误做法**             |
| ---------- | ------------------------------------------------------------ | ------------------------ |
| 速度       | 第一次下雨或刚停雨时最滑，需减速至 **5–10 mph 以下于限速**。 | 雨后马上加速。           |
| 灯光       | 必须开 **近光灯 (low beam)**。                               | 用远光灯（反光更严重）。 |
| 车距       | 比平时 **增加一倍**。                                        | 仍保持三秒。             |
| 雨刷效果差 | 更换或停车等待。                                             | 继续行驶。               |

------

## **🌫️ 4️⃣ 雾天驾驶（Fog）**

| **场景** | **正确做法**                   | **错误做法**           |
| -------- | ------------------------------ | ---------------------- |
| 浓雾行车 | 开 **近光灯 + 雾灯**；减速。   | 用远光（会反射更亮）。 |
| 雾太浓   | 打开应急灯，**驶离道路停下**。 | 停在车道中间。         |
| 灯光规则 | 1000 ft 内能见度不足须开灯。   | 忘记开灯。             |

------

## **❄️ 5️⃣ 冰雪与山路（Ice, Snow, and Mountain Driving）**

| **场景**         | **正确操作**                     | **错误操作**     |
| ---------------- | -------------------------------- | ---------------- |
| 冰面打滑         | 使用低速档（low gear），慢刹车。 | 高速/突然刹车。  |
| 下坡             | 提前降档，利用发动机制动。       | 空档滑行。       |
| 上坡打滑         | 缓慢加速、避免原地打滑。         | 猛踩油门。       |
| 轮胎链           | 安装在驱动轮。                   | 装错轮或未紧固。 |
| 黑冰 (black ice) | 在阴影/桥上更常见。              | 忽视结冰风险。   |

------

## **🌬️ 6️⃣ 强风驾驶（High Winds）**

| **正确做法**                     | **错误做法**     |
| -------------------------------- | ---------------- |
| 双手握稳方向盘、减速、保持车距。 | 急打方向或加速。 |

------

## **🔦 7️⃣ 夜间驾驶（Night Driving）**

| **要点** | **正确做法**                                             | **错误做法**       |
| -------- | -------------------------------------------------------- | ------------------ |
| 灯光使用 | 1000 ft 内必须开灯；500 ft 内遇车调近光。                | 忘关远光。         |
| 速度     | 需随可见范围调整；“不要超过灯光照射范围内可刹停的距离”。 | 看不清仍维持高速。 |

------

## **⚙️ 8️⃣ 车辆安全与应急设备（Vehicle Control Hidden Rules）**

| **场景**           | **法规要点**                                                 |
| ------------------ | ------------------------------------------------------------ |
| **轮胎胎纹**       | 法定最小深度 1/32 inch；用 Lincoln 硬币测（头顶露出 = 不达标）。 |
| **轮胎气压**       | 应冷车测量，气压低易 Hydroplane。                            |
| **雨刷/除雾器**    | 雨中必须有效；失效属车辆缺陷违法。                           |
| **Cruise Control** | 禁止在湿滑或结冰路面使用。                                   |

------

## **🧠 小结记忆法**

> 🚗「**三降一稳**」：减速、降档、增距、稳方向。
>
> 💧「**35 mph = 水滑线**」：超过即失抓地。
>
> 🌫️「**雾中近光不远光**」。
>
> ❄️「**低速挡控下坡、黑冰在桥上**」。
>
> 🌬️「**双手稳方向、提前减速**」。

## **🚗 一、速度类隐藏知识点**

| **知识点**                   | **关键数字**                                                 | **说明 / 规律**                                              | **考点**                                                |
| ---------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------- |
| **Hydroplaning（车胎水滑）** | ⚠️ **> 35 mph**                                               | 超过35 mph后轮胎无法有效排水，可能失去抓地。                 | “At what speed can hydroplaning begin?” ✅ 35 mph 左右。 |
| **视线与速度匹配**           | “Don’t drive faster than you can stop within the distance you can see ahead.” | 意为**速度必须匹配照明范围**。夜间灯光照射不到的距离不能超速。 | 夜间驾驶题。                                            |
| **雾天安全速度**             | 减至标限速 – 5~10 mph                                        | 雾天/雨天“Adjust your speed 5–10 mph below the limit”。      | Weather题。                                             |
| **下坡控速**                 | “Use a lower gear.”                                          | 刹车易过热，应**提前降档**控制速度。                         | 山路题。                                                |
| **牵引拖车限速**             | ≤ 55 mph                                                     | 所有 towing vehicles (含房车拖车)。                          | 车道和限速题。                                          |
| **住宅区默认限速**           | 25 mph                                                       | 无标志区域默认。                                             | 基础法规。                                              |
| **巷道 / 盲交 / 铁路**       | 15 mph                                                       | 特殊低速。                                                   | 低速陷阱题。                                            |

# 各种 Law - 隐性驾驶原则 / 特殊规则

「跟随车流速度」的确不是幻觉：

它确实存在，但它**不是一条单独的法律 (law)**，而是加州 DMV 驾驶考试里经常与 **Basic Speed Law** 一起出的“**隐性原则类规则**”。

这些属于「**Special Driving Principles / Laws of the Road**」系列：

它们在手册中散落各章（限速、车距、超车、路况），经常被出题为陷阱题。

## **⚖️ 一、The** **Basic Speed Law**

> **You must never drive faster than is safe for current conditions.**

| **项目**         | **内容**                                                     |
| ---------------- | ------------------------------------------------------------ |
| ✅ **核心原则**   | 不论限速标志是多少，都必须以**路况安全速度**为准。           |
| 🚦 **示例**       | 限速 55 mph → 雾、雪、弯道、交通拥堵时开 45 mph 仍可能被罚。 |
| ⚠️ **常见陷阱题** | “If you drive slower than the posted limit but unsafe for conditions, you can be cited.” ✅ True |
| 📘 **出处**       | CVC §22350                                                   |

------

## **🛣️ 二、The** **Prima Facie Speed Law**

> “When no speed limit signs are posted, specific default limits apply.”

| **场景**                                                | **默认限速**          |
| ------------------------------------------------------- | --------------------- |
| 住宅区、商业区（business/residential area）             | 25 mph                |
| 学校区域（within 500–1000 ft of a school）              | 25 mph                |
| 小巷 (alley)                                            | 15 mph                |
| 铁路交叉口、盲交路口、山路转弯处（visibility ≤ 100 ft） | 15 mph                |
| Two-lane undivided highway                              | 55 mph                |
| 多车道高速 (Freeway)                                    | 65–70 mph（视段而定） |

📘 “Prima facie” 是拉丁语，意思是「在没有特别标志的情况下默认生效」。

------

## **🚗 三、The** **Minimum Speed Law**

> **You may not drive so slowly that you block or impede normal traffic flow.**

| **核心要点**       | **说明**                                      |
| ------------------ | --------------------------------------------- |
| 🚫 不得低速阻碍交通 | 若车流明显比你快，应靠右让行。                |
| 🚓 若造成堵塞       | 可被罚 “impeding traffic”（哪怕你低于限速）。 |
| 🚗 适用场景         | 上坡、拖车、重载、或观光慢行。                |
| 📘 出处             | CVC §22400                                    |

------

## **🌀 四、The** **Flow of Traffic Principle**

> **Drive at the speed of traffic — but never faster than the speed limit.**

| **项目**       | **内容**                                                     |
| -------------- | ------------------------------------------------------------ |
| ✅ **正确做法** | 与车流保持相近速度，以防被追尾或造成危险。                   |
| ⚠️ **错误理解** | “随流行驶” ≠ “超速跟流”。超过限速依然违法。                  |
| 💡 **出题陷阱** | 题干常写：“You should drive with the flow of traffic, even if it means exceeding the limit.” → ❌ False |
| 📘 **解释**     | DMV 手册中描述为 “Keep pace with traffic, but never exceed the speed limit.” |

------

## **⚠️ 五、The** **Basic Following Distance Rule**

> **Keep a space cushion of at least 3 seconds.**

| **条件**           | **距离**                                          |
| ------------------ | ------------------------------------------------- |
| 正常天气           | ≥ 3 秒                                            |
| 雨天 / 雾天 / 夜间 | ≥ 4 秒                                            |
| 拖车 / 大货车      | ≥ 5 秒                                            |
| 山路 / 结冰        | ≥ 6 秒                                            |
| 📘 说明             | 属于防御驾驶（Defensive Driving）的基础原则之一。 |

------

## **⛔ 六、The** **Right-of-Way Principles**

> “The law does not give you the right-of-way; it only tells you when you must yield.”

| **原则**                 | **含义**                                       |
| ------------------------ | ---------------------------------------------- |
| **必须让行而非抢行**     | 即使“理论上”你有优先通行权，也要避让以防事故。 |
| **盲区、让行标志、行人** | 永远礼让行人，即使信号灯绿。                   |
| **四向停（4-way stop）** | 先停者先行；同时停者让右边先行。               |

------

## **🧊 七、The** **Safe Following & Passing Laws**

| **项目**             | **规则**                            | **限制**           |
| -------------------- | ----------------------------------- | ------------------ |
| **Passing on Left**  | Only when safe and clear.           | 不得越双实线。     |
| **Passing on Right** | 仅在前车左转时、或多车道同向时。    | 不得驶离路面肩部。 |
| **Don’t Tailgate**   | 保持 space cushion，防止 rear-end。 | 3 秒法则。         |

------

## **🌧️ 八、The** **Too Fast / Too Slow Conditions**

| **场景**     | **可被罚的情况**           | **原因**               |
| ------------ | -------------------------- | ---------------------- |
| 雾中开太快   | ✅ Too fast for conditions  | 违反 Basic Speed Law   |
| 大雾太慢挡路 | ✅ Impeding traffic         | 违反 Minimum Speed Law |
| 跟流超速     | ✅ 违法（限速上限绝对有效） | 无论其他车快慢         |
| 下坡溜车     | ✅ 超速                     | 限速包括坡道           |

------

## **🚨 九、The** **Move Over Law**

> If you see flashing lights ahead (police, ambulance, tow truck), you must move over or slow down.

| **项目**                                                     | **内容**   |
| ------------------------------------------------------------ | ---------- |
| ✅ Move over one lane **if safe**, or slow down significantly. |            |
| 🚓 Applies to emergency vehicles on **shoulder or roadside.** |            |
| 📘 来源                                                       | CVC §21809 |

------

## **🚦 十、The** **“Drive to the Right” Law**

> On highways with two or more lanes, drive in the **right lane** except to pass.

| **原则**                   | **内容**                 |
| -------------------------- | ------------------------ |
| Keep right except to pass. | 除非超车，否则走右车道。 |
| Slower traffic keep right. | 低速车靠右，不占快车道。 |

------

## **🧩 十一、The** **Two-Second Reaction Principle**

> Always leave enough space to react to the unexpected.

| **事项**                            | **时间 距离**                                 |
| ----------------------------------- | -------------------------------------------- |
| 平均反应时间 | ≈ 1 秒 |
| 平均反应距离 | ≈ 55 ft @ 55 mph |
| 停车距离 | ≈ 400 ft @ 55 mph |
| 口诀 | “One second think, three seconds act.” |

------

## **🌟 十二、其他容易忽略的「特别原则」**

| **名称**                            | **核心内容**                                 |
| ----------------------------------- | -------------------------------------------- |
| **Anti-Gridlock Law**               | 禁止在路口中间等待（会阻塞交通）。           |
| **California’s “Keep Clear” Zones** | 白色“KEEP CLEAR”字样区禁止停车。             |
| **Safety Belt Law**                 | 所有人必须系安全带；16岁以下乘客责任归司机。 |
| **Headlight Law**                   | 夜间/雾中能见度<1000ft 必开灯。              |
| **Handheld Device Ban**             | 驾驶中不得手持电子设备。                     |
| **Construction Zone Fines**         | 罚款加倍。                                   |

------

## **🧠 一句话记忆法**

> - 🛣️ **Basic Speed Law:** 永远随“路况”，不是标志。
>
> - 🌀 **Flow of Traffic:** 可随流，但绝不超速。
>
> - 🧍 **Minimum Speed Law:** 不可慢到妨碍别人。
>
> - ⚖️ **Right-of-Way:** 让行不是权利，是责任。
>
> - 🚓 **Move Over:** 看见闪灯 → 换道或减速。

# **物理与驾驶常识陷阱**

*(Physical & Environmental Driving Logic Traps — 2025 California Edition)*

------

## **🧊 一、道路冻结与打滑（Friction & Temperature）**

| **场景**                           | **正确现象**                           | **典型误解**     | **出题陷阱**                                               |
| ---------------------------------- | -------------------------------------- | ---------------- | ---------------------------------------------------------- |
| **桥梁 / 高架 (Bridge, Overpass)** | 🌡️ 冻得最快，因为上下都暴露在冷空气中。 | 以为地面先冻。   | “Which roadways freeze first when wet?” ✅ Bridges.         |
| **隧道 (Tunnel)**                  | 🔥 最不容易冻结，被地面包围保温。       | 以为阴冷易结冰。 | 错选 Tunnels。                                             |
| **黑冰 (Black Ice)**               | ❄️ 无色透明冰层，常出现在桥上、阴影处。 | 以为冰总能看见。 | “Ice may be invisible and extremely slippery.”             |
| **打滑时方向盘操作**               | 应顺着**车尾方向**打方向盘。           | 以为要反打。     | “Turn in the direction of the skid.” ✅                     |
| **第一次下雨**                     | 最滑，因为路面油污被雨水浮起。         | 以为雨中间更滑。 | “The first rain after dry weather is the most slippery.” ✅ |

------

## **🌧️ 二、水滑现象 Hydroplaning（轮胎浮水）**

| **原理**                                        | **说明** |
| ----------------------------------------------- | -------- |
| 💧 轮胎无法及时排出水 → 浮在水面上，失去抓地力。 |          |
| 🚗 **开始速度：35 mph 起**，高速时更危险。       |          |
| 🧠 正确应对：**松油门、稳方向**，不要刹车。      |          |
| ❌ 错误反应：猛踩刹车 / 打方向。                 |          |

📘 考题：

> “At about what speed can tires start hydroplaning?”
>
> ✅ *At about 35 mph.*

------

## **⚙️ 三、刹车与控制（Braking Physics）**

| **场景**           | **正确操作**                  | **原理**         | **常错选项**                                  |
| ------------------ | ----------------------------- | ---------------- | --------------------------------------------- |
| **ABS 防抱死系统** | 一直踩住刹车（do not pump）。 | ABS 会自动点刹。 | “Pump the brakes.” ❌                          |
| **无 ABS**         | 需轻踩点刹。                  | 防止锁轮。       | “Apply firm continuous pressure.” ❌           |
| **冰雪上急刹**     | 前轮锁死 → 失去转向。         | 无摩擦控制方向。 | “Continuous hard braking locks the wheels.” ✅ |
| **下坡长刹**       | 会导致刹车过热。              | 动能转热能。     | 正确：用低速档减速。                          |

------

## **🔥 四、速度与距离（Speed vs. Stopping Distance）**

| **场景**       | **平均数值**                                                 | **含义**                 |
| -------------- | ------------------------------------------------------------ | ------------------------ |
| **反应时间**   | ≈ 1 秒                                                       | 从看到危险到开始刹车。   |
| **反应距离**   | ≈ 55 ft @ 55 mph                                             | 人的延迟导致行驶距离。   |
| **制动距离**   | ≈ 300 ft @ 55 mph                                            | 刹车后到完全停下的距离。 |
| **总停止距离** | ≈ 400 ft @ 55 mph                                            | 反应 + 制动。            |
| **视线规则**   | “Don’t drive faster than you can stop within your visible distance.” |                          |

📘 常考句型：

> “How far will your vehicle travel before you stop?”
>
> ✅ *About 400 ft when traveling at 55 mph.*

------

## **🌫️ 五、天气与能见度（Visibility & Light）**

| **场景**               | **正确做法**               | **错误做法**         |
| ---------------------- | -------------------------- | -------------------- |
| **雾天 / 雨天 / 雪天** | 用 **近光灯 (low beam)**。 | 开远光（反射炫目）。 |
| **浓雾中停靠**         | 必须驶出车道。             | 停在车道中危险。     |
| **雨天速度**           | 比限速低 5–10 mph。        | 按限速甚至更快。     |
| **能见度 ≤1000 ft**    | 必须开灯。                 | 忘开灯违法。         |

📘 考题：

> “When it is foggy, drive slower and use your low-beam headlights.” ✅

------

## **🌡️ 六、温度与轮胎（Tire Pressure & Traction）**

| **项目**     | **正确值**                        | **考点**            |
| ------------ | --------------------------------- | ------------------- |
| **胎纹深度** | ≥ 1/32 inch（Lincoln 硬币头盖住） | 否则易 Hydroplane。 |
| **胎压低**   | 增加磨损 + 降低抓地力。           | 夏天易爆胎。        |
| **高温天气** | 路面软化 → 增刹车距离。           | 防御性驾驶。        |

------

## **🏔️ 七、坡道与重力（Grade & Momentum）**

| **场景**         | **原理**                              | **正确操作**          |
| ---------------- | ------------------------------------- | --------------------- |
| **下坡刹车过热** | 长时间踩刹 → 热衰减。                 | 提前降档 (Low gear)。 |
| **上坡停车起步** | 防止后滑。                            | 使用手刹辅助起步。    |
| **坡道停车方向** | 上坡无缘：靠左；上坡有缘/下坡：靠右。 | “上左下右无右”口诀。  |
| **停车转轮**     | 朝向路沿或反方向，防止溜车。          | 考题常考。            |

## **🚨 八、车身惯性与碰撞物理（Inertia & Force）**

| **现象**                    | **原理**                | **典型错误**       |
| --------------------------- | ----------------------- | ------------------ |
| **突然刹车时身体前倾**      | 惯性保持运动方向。      | 认为是安全带原因。 |
| **侧撞时甩出车外**          | 没系安全带 → 惯性飞出。 | 以为气囊足够保护。 |
| **速度翻倍 → 撞击能量四倍** | 动能 = ½mv²。           | 误认为两倍。       |

📘 口诀：

> “双倍速度 → 四倍冲击。”

## **🚗 九、牵引 / 拖车物理限制（Traction & Weight）**

| **项目**       | **限制**               | **原理**         |
| -------------- | ---------------------- | ---------------- |
| **牵引车限速** | ≤ 55 mph               | 牵引增加惯性。   |
| **转弯半径**   | 更大（需更早打方向）。 | 拖车会向内侧切。 |
| **下坡牵引**   | 提前减速，低档行驶。   | 重力加速度。     |

## **⚡ 十、风阻与空气动力学（Air & Motion）**

| **场景**       | **正确做法**       | **原理**               |
| -------------- | ------------------ | ---------------------- |
| **强风天气**   | 双手稳方向、减速。 | 侧风影响车体平衡。     |
| **开窗驾驶**   | 阻力上升、油耗高。 | 速度 > 45 mph 更明显。 |
| **卡车旁行驶** | 避开盲区与气流。   | 大车气流会吸/推。      |

## **🧱 十一、摩擦与路面种类（Traction Types）**

| **路面类型**           | **抓地力等级** | **特点**           |
| ---------------------- | -------------- | ------------------ |

## **📘 附加：法规 vs 路标易混概念总表**


| **概念类型**                         | **实际含义**                                            | **容易混淆点**                 |

| 左/右转            | **closest-to-closest** | 左转：最左→最左；右转：最右→最右。                           | 直接跨入第二/第三车道。      |
| 多车道左转         | **按地面箭头**         | 多左转车道：各走各道，不可转向中跨线。                       | 认为可任选任意目标车道。     |
| 中央双黄虚实左转道 | **center turn lane**   | 仅用于左转/掉头；在道内行驶≤300 ft。                         | 当作超车或直行车道。         |
| 掉头               | **U-turn规则**         | 允许处需确保安全；**禁**：学校区、铁路道口、高速、标“NO U-TURN”。 | 商业区随意掉头；未判定视距。 |

## **七、行人与自行车**

| **类别**     | **考点**              | **正确做法 / 答案**                                    | **常见错点**                   |
| ------------ | --------------------- | ------------------------------------------------------ | ------------------------------ |
| 行人优先     | **斑马线/无控交叉口** | 行人一律优先（含未划线但法定人行横道）。               | 以为没画线就不让。             |
| 自行车道     | **右转先并入**        | 右转前**先进入自行车道**再转，注意盲区；禁止占用停车。 | 直接从机动车道右转、压线切入。 |
| 与骑行者距离 | **3 英尺法**          | 超车时至少 3 ft 横向距离。                             | 近距离挤压逼停。               |

## **八、停车距离/坡道停车**

| **类别** | **考点**               | **正确做法 / 答案**                                          | **常见错点**             |
| -------- | ---------------------- | ------------------------------------------------------------ | ------------------------ |
| 距离     | **消防栓/铁路/公交站** | 消防栓 **15 ft**；铁路道口 **50 ft** 外；公交站牌 **30 ft** 外。 | 停在消防栓边“人不离车”。 |
| 坡道     | **车轮朝向**           | Uphill有路缘：前轮**向左**；无路缘：**向右**；Downhill：**向右**；拉手刹。 | 一律朝右/忘拉手刹。      |

## **九、施工区/紧急车辆**

| **类别** | **考点**          | **正确做法 / 答案**                                   | **常见错点**            |
| -------- | ----------------- | ----------------------------------------------------- | ----------------------- |
| 施工区   | **Fines doubled** | 减速、服从旗手/标志，罚款加倍。                       | 当成仅建议。            |
| 紧急车辆 | **让行规则**      | 听到警笛/看到警灯：**靠右停下**；必要时越线让出空间。 | 停在路口中央/继续行驶。 |

## **十、酒驾/药驾与灯光**

| **类别** | **考点**      | **正确做法 / 答案**                   | **常见错点**            |
| -------- | ------------- | ------------------------------------- | ----------------------- |
| DUI      | **药驾=违法** | 合法处方若影响驾驶仍属违法。          | 以为非酒精就不算DUI。   |
| 灯光     | **雾天/夜间** | 雾中用**近光**；夜间1000 ft内须开灯。 | 用远光穿雾/阴天不开灯。 |

## **小口诀**

- **路缘五色**：红禁、黄装、白上下、绿限时、蓝残障。
- **三类 15 mph**：巷道、盲交、铁路视距差。
- **转弯**：左进左、右进右；多车道按箭头。
- **坡停车**：上左下右，有无路缘要分清。
- **施工**：罚双倍；**耳机**：双耳禁。
