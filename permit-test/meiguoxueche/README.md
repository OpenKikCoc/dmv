**111 在酷熱天氣將6歲以下兒童留在車中且無人看護是否違法？**
- <span style="color:#16a34a">✅ 是的，即使兒童巳經被妥善的安置在兒童安全座椅上。</span>
- 是的，即便兒童由12歲以上者照看也不行。
- 不是，只要車鑰匙留在車內就不違法。
- 不是，因為加州氣候炎熱。

> 必须选一个既表示“是真的”，又没有加奇怪额外条件的：
>	1.	✅ 真的，即使儿童已经在安全座椅上
>	2.	❌ 真的，即便由 12 岁以上者照看也不行（这句就和法律真实条文对不上）
>	3.	❌ 不是真的，只要钥匙留在点火器里就不违法（离谱）
>	4.	❌ 不是真的，因为加州常年炎热所以不会有这种法律（更离谱）
>
> 所以只能选 1，因为它是：
>
> “是真的” + “不会改动法律原句意思”。
