# 路考指令 中英文对照

## 【考前检查】 (Pre-Drive Checklist)

| 英文指令 | 中文含义 | 操作 / 备注 |
| :--- | :--- | :--- |
| (Start Engine) | 点火 | **Turn off** -> **Turn on** (转两下) -> **点着火** |
| Left Turn Signal | 打开左转灯 | |
| Right Turn Signal | 打开右转灯 | |
| Step on the Brake | 踩刹车 | |
| Headlights | 车头灯 | |
| Brakelights | 刹车灯 | |
| Horn | 按喇叭 | |
| Glove Box | 手套箱 | |
| Rear view mirrors | 后视镜 | |
| Arm signals | 手臂信号 | **Left** (左转): 手臂水平向左<br>**Right** (右转): 手臂向上<br>**Stop** (停车): 手臂向下<br>(注意弯着手，不要伸直) |
| Show me the emergency brake | 指出手刹 | **上拉**: 开启 (踩住刹车 + 向上拉)<br>**向下**: 松手刹 (踩住刹车 + 向下压) |
| Show me the defroster | 指出除雾 | 记得有两个按钮 |
| Show me the emergency flasher | 指出双闪灯 | (Hazard flashes) |
| Show me the headlights | 指出车头灯开关 | **Low beams** (近光灯): 转到“水母”图标<br>**High beams** (远光灯): 转到水母后，往前拨 |
| Show me the windshield wipers | 指出雨刷开关 | 右边向上拨，再向下拨还原 |

## 【路考过程】 (Driving Test Maneuvers)

| 英文指令 | 中文含义 | 备注 |
| :--- | :--- | :--- |
| Turn left/right | 左/右转 | |
| At the corner | 在转角处 | |
| At the traffic light | 在红绿灯处 | |
| At the intersection | 在十字路口 | |
| Change lane to the left/right | 左/右换道 | |
| Stay in this lane | 在这条道上行驶 | |
| Go straight | 直行 | |
| Pull over and stop | 路边停车 | (Pull over parking) |
| Back up | 倒车 | |
