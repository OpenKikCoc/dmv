# **加州 DMV 路考注意点 🔥**

---

- 考官什么时候上车？
    - 考前 check 的时候，考官在车外走动，检查转向灯、刹车灯是否正常，考前 check 结束后，考官上车。
    - 考完到停车位之后，停车、算分，考官下车。
- 上车调整座椅，**脚跟要靠着地上**。
- 上车之后要一直踩着刹车吗？  
    - **不用**，只有启动油门的时候和之后。

---

## **二、注意点**

- 观察路况，礼让行人。
- 注意 **stop 标志**。
- 红绿灯过不了就停下来。
- 小路口，黄色实线 + 黄色虚线是待转区。
- 要多多看路牌的限速，要在上下 3 以内。
  有时大路会有 40 的路牌，有 30 的路牌。
- <span style="color:red">路考时，考官没被指令就是直行。</span>
- 学会看限速。
- 左转可以慢慢开到路口的 1/3（**不能压双黄线**）。
- 入弯减速：**大弯 16、小弯 13**。
- 路口前面堵车，不要出去，前后都不能停在路口当中。
- 考场那边车比较宽，直行的时候靠左边，右转的时候要靠右边（平时是停车的那一侧）（打转向灯变道）。
- 红灯右转的话，如果视线不好 + 车多，可以选择绿灯再右转。
- 过路口左转，**缓慢 1/3**，<span style="color:red">注意对向车 + 行人 + 红绿灯</span>，如果红绿灯要变了，要抓紧在对向车道的最后一辆车过去之后，过去。  
    - 人行道上的人，过掉 3/4 马路再走。
- 转弯，**速度 + 转向配合好**，不要太慢太快。
- 如果自己路权高（右转向箭头灯，对方单向 stop 等），不要太谨慎，**加速过**。

---

## **五、限速（Speed limit）**

- DMV 内部不超过 **10**。
- 住宅区域 **25**（一般 22–28 区间）。
- 学校：看上面限速标志。
- 大路：  
  - **35（32–38）**  
  - **40（37–43）**
- 小弯：**12–13**（一般右转）。
- 大弯：**13–16**（一般左转）。
- <span style="color:red">hump（居民区凸起来的一块）：15 以下（看地上白色倒着的三角，或者旁边有 hump 标）。</span>

---

## **六、指示标变道 / 转向**

### **变道**

- 步骤：  
  **转向灯 + 侧视镜 + 盲点 + 变道 <span style="color:red">(不减速!)</span> + 灯还原**

### **路口转弯**

- 先往左/右靠（<span style="color:red">变道，注意左右看</span>），过转向前一左一右看。

- **右转：**  
  - 虚线处进入最右侧车道。  
  - 转后，拐到离我最近的车道（有自行车道不拐进去，有自行车标，靠一点）。

- <span style="color:red">出弯就要提速。</span>

- **绿灯右转：**  
  - <span style="color:red">绿灯直接转（看行人）</span>。  
  - <span style="color:red">红灯要往左看有没有车</span>。

- **左转：**  
  - 给直行 + 右转车辆让行 (不给左转车让行)。  
  - (看左，看前面)

---

## **七、stop sign**

- 无 stop sign：左右看，开车不减速。
- 有 stop sign：  
  - **stop sign 线前面停一下**。  
  - **4 个 stop sign：先到先走。**  
  - **2 个 stop sign（要让横向车辆先走）**：停下，左右看，缓行，左右看，（没车）直行。

- 和右边停的车保持距离走。

---

## **八、路考考点**

- **出考点：**  
  中间停下，左右看，出去。

- **住宅区：**  
  stop sign、限速、左右转。

- **大马路：**  
  变道、左右转。

- **直行长的马路：**  
  <span style="color:red">pull over (靠路边停，靠路边 30cm)，宁愿远也不要近。</span>

### **pull over 细节**

<span style="color:red">
pull over：路边停车，30cm，两条大腿的距离，就是白色那条窄一点点（不能停到车库门口、消防栓门口，不能挡着别人）。
</span>

### **倒车（reverse / back up）**

<span style="color:red">
倒车（reverse / back up）：前后左右看，安全，再倒档；右手搭在右边副驾驶位，倒车的时候看右侧后视镜，和向右看。
</span>

### **开出来（pull out）**

<span style="color:red">
开出来：pull out（转向灯 + 左看后视镜 + 回头看）。
</span>

- 停车的时候和前面保持一个车位的距离：
  - 坐姿低：看到完整的后轮胎；
  - 坐姿高：看到地面。

### 其他

- 避让右边路上的人，如果左边要压线，先要打转向灯。

- 右转：虚线或没有线靠上去，实线不靠进去。  
  实际考场：35 / 40 限速较多。

- **first left lane**：从左往右数第一个车道。  
- **first right lane**：从右往左数第一个车道。

- 校车（竖起 stop 的时候）、消防车靠边 / 就低停车。

---

## **九、意识**

- 跟上车流，融入车流。
- <span style="color:red">路权最大。</span>
- <span style="color:red">箭头转向灯路权最大，直接走。</span>
- 提前注意看 stop sign。

# 自己总结

注意事项

1.注意限速路标。绝不可以超速，否则直接fail，速度不能太快也不能太慢，都有可能导致不及格
2.左右转弯前，一定要先看上方后照镜一车侧后照镜 打灯 明显转头看 再换线
  左右转时，注意不要压到白线
3.左右转头看死角的动作请一定要明显。（把头转过肩最保险）
4.红灯右转一定要在线后先踩刹2-3秒，再往前左右看，确认没有行人与左边直行车及前面左转车的安全情况下才可转弯，如有行人一定要等行人通过后才可转弯
5.确认是否有 NotTurnonRed的警告牌，有的话就不可右转。
6.遇到STOP标示的重点提醒，STOPsign 一定要煞死3秒再起步
  若遇到4-WaySTOP，表示四个路口都有STOP标志，先到先走，不要抢快
  若遇到自己路口有STOP，但旁边路口没有，则在STOP停车三秒后，先露头开出，然后再煞一次，并左右再次观看确认没车才可行走。
7.PullOver路边停车
  路边停车时一定要打右转灯；完成后从路边出来时，一定要打左转灯
  路边停车时请与路边保持恰当距离，不要太近也不要太远
  若遇到校车，双向的车都要停下
  若遇到鸣笛的警车或消防车，双向的车都要立即向右边停靠。
8.倒车
  在路边停车后，考官会请考生直接进行倒车动作，倒车前，先前后左右看再倒车，须保持直线倒车。

1️⃣ 礼让行人。在看到行人的时候一定一定要刹车停稳，不然就直接挂了，尤其是在DMV内以及附近。

2️⃣ 转弯幅度。在Sanjose的路线上路都比较窄，也几乎没有没有非机动车，所以左转只要车头出了停止线就可以打方向了，右转在停止线处就应该已经打了一点方向（做好shoulder check）。

3️⃣ Stop Sign。之前我以为只要停稳再出发就行了，经过教练的指导我才知道原来还有是否是allway的区别。如果是all way的在停稳以后【确认其他方向都没车以后】就应该毫不犹豫地起步出发，不要再有unnecessary stop；如果不是allway的话一定车头出去以后再左右看一下有没有横向的车过来，因为对方是没有Stop Sign的。而且不是每个都标记了是否allway，一定自己看一下横着的这条路有没有Stop Sign。

4️⃣ 限速。路考的时候不仅有最高限速，也不能开得太慢，例如30限速就要控制在25-30之间，【在DMV里控制在10以内】。

5️⃣ 驾驶姿势。路考时一定要让考官清楚地看到你的动作，所以在转弯或者变道的时候，所有的动作都是往前坐直的，包括左右看的时候要把头往前探，shoulder check的幅度大一点。总而言之一定要让考官感觉到你开车的过程中一直非常注意路况。

6️⃣ 底层逻辑。我认为在这里路考的底层逻辑就是要胆大心细，对路况时刻保持警惕，但是该走就走，该停就停。
