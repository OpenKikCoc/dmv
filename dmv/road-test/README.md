# 注意事项

# **加州 DMV 路考注意点 🔥**

---

- 考官什么时候上车？
    - 考前 check 的时候，考官在车外走动，检查转向灯、刹车灯是否正常，考前 check 结束后，考官上车。
    - 考完到停车位之后，停车、算分，考官下车。
- 上车调整座椅，**脚跟要靠着地上**。
- 上车之后要一直踩着刹车吗？  
    - **不用**，只有启动油门的时候和之后。

---

## **二、注意点**

- 观察路况，礼让行人。
- 注意 **stop 标志**。
- 红绿灯过不了就停下来。
- 小路口，黄色实线 + 黄色虚线是待转区。
- 要多多看路牌的限速，要在上下 3 以内。  
  有时大路会有 40 的路牌，有 30 的路牌。
- <span style="color:red">路考时，考官没被指令就是直行。</span>
- 学会看限速。
- 左转可以慢慢开到路口的 1/3（**不能压双黄线**）。
- 入弯减速：**大弯 16、小弯 13**。
- 路口前面堵车，不要出去，前后都不能停在路口当中。
- 考场那边车比较宽，直行的时候靠左边，右转的时候要靠右边（平时是停车的那一侧）（打转向灯变道）。
- 红灯右转的话，如果视线不好 + 车多，可以选择绿灯再右转。
- 过路口左转，**缓慢 1/3**，<span style="color:red">注意对向车 + 行人 + 红绿灯</span>，如果红绿灯要变了，要抓紧在对向车道的最后一辆车过去之后，过去。  
    - 人行道上的人，过掉 3/4 马路再走。
- 转弯，**速度 + 转向配合好**，不要太慢太快。
- 如果自己路权高（右转向箭头灯，对方单向 stop 等），不要太谨慎，**加速过**。

---

## **三、考前 check（车外 / 车内检查）**

- **turn off**
- **turn on**（转两下）
- **点着火**

- **step on the brake：刹车**

- **emergency brake / handbrake（手刹）**  
  - 上拉手刹：开启手刹（踩住刹车 + 向上拉）。  
  - 向下：松手刹（踩住刹车 + 向下压）。

- **headlight 前照灯（包括近光灯 + 远光灯）**
  - **low beams 近光灯**（转到“水母”图标）
  - **high beams 远光灯**（转到水母后，往前拨）

- **window wipes / windshield wiper 雨刮器**  
  - 右边向上拨，再向下拨还原（不用向下拨，自动还原）。

- **emergency flashes / hazard flashes 应急闪光灯（双闪）**

- **defroster 除雾开关**（记得有两个按钮）

- **hand signal（注意是弯着手，不要伸直）**  
- arm signal：  
  - **left（左转）** – 手臂水平向左  
  - **right（右转）** – 手臂向上  
  - **stop（停车）** – 手臂向下

## **四、路考指示（口令）**

- **pull over parking：路边停车**
- **back up：倒车**

---

## **五、限速（Speed limit）**

- DMV 内部不超过 **10**。
- 住宅区域 **25**（一般 22–28 区间）。
- 学校：看上面限速标志。
- 大路：  
  - **35（32–38）**  
  - **40（37–43）**
- 小弯：**12–13**（一般右转）。
- 大弯：**13–16**（一般左转）。
- <span style="color:red">hump（居民区凸起来的一块）：15 以下（看地上白色倒着的三角，或者旁边有 hump 标）。</span>

---

## **六、指示标变道 / 转向**

### **变道**

- 步骤：  
  **转向灯 + 侧视镜 + 盲点 + 变道 <span style="color:red">(不减速!)</span> + 灯还原**

### **路口转弯**

- 先往左/右靠（<span style="color:red">变道，注意左右看</span>），过转向前一左一右看。

- **右转：**  
  - 虚线处进入最右侧车道。  
  - 转后，拐到离我最近的车道（有自行车道不拐进去，有自行车标，靠一点）。

- <span style="color:red">出弯就要提速。</span>

- **绿灯右转：**  
  - <span style="color:red">绿灯直接转（看行人）</span>。  
  - <span style="color:red">红灯要往左看有没有车</span>。

- **左转：**  
  - 给直行 + 右转车辆让行 (不给左转车让行)。  
  - (看左，看前面)

---

## **七、stop sign**

- 无 stop sign：左右看，开车不减速。
- 有 stop sign：  
  - **stop sign 线前面停一下**。  
  - **4 个 stop sign：先到先走。**  
  - **2 个 stop sign（要让横向车辆先走）**：停下，左右看，缓行，左右看，（没车）直行。

- 和右边停的车保持距离走。

---

## **八、路考考点**

- **出考点：**  
  中间停下，左右看，出去。

- **住宅区：**  
  stop sign、限速、左右转。

- **大马路：**  
  变道、左右转。

- **直行长的马路：**  
  <span style="color:red">pull over (靠路边停，靠路边 30cm)，宁愿远也不要近。</span>

### **pull over 细节**

<span style="color:red">
pull over：路边停车，30cm，两条大腿的距离，就是白色那条窄一点点（不能停到车库门口、消防栓门口，不能挡着别人）。
</span>

### **倒车（reverse / back up）**

<span style="color:red">
倒车（reverse / back up）：前后左右看，安全，再倒档；右手搭在右边副驾驶位，倒车的时候看右侧后视镜，和向右看。
</span>

### **开出来（pull out）**

<span style="color:red">
开出来：pull out（转向灯 + 左看后视镜 + 回头看）。
</span>

- 停车的时候和前面保持一个车位的距离：
  - 坐姿低：看到完整的后轮胎；
  - 坐姿高：看到地面。

### 其他

- 避让右边路上的人，如果左边要压线，先要打转向灯。

- 右转：虚线或没有线靠上去，实线不靠进去。  
  实际考场：35 / 40 限速较多。

- **first left lane**：从左往右数第一个车道。  
- **first right lane**：从右往左数第一个车道。

- 校车（竖起 stop 的时候）、消防车靠边 / 就低停车。

---

## **九、意识**

- 跟上车流，融入车流。
- <span style="color:red">路权最大。</span>
- <span style="color:red">箭头转向灯路权最大，直接走。</span>
- 提前注意看 stop sign。

---

## **十、一下子就挂的行为**

<span style="color:red">
- 速度过慢或过快  
- 撞到路牙子  
- 压到双黄线  
- 开错车道（最左边、最右边）  
- 和车争路权  
- 不让行人  
- 变道不看侧视镜、盲点  
- stop sign 不停下来  
</span>

(压虚线为小错)
